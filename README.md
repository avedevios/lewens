Lewens is a cross-platform mobile application targeting iOS and Android, focusing on seamless integration with Keycloak and high-performance user experience.

## 📂 Project Structure

- **[lewens-ios/](file:///Users/ake11a/Swift/Lewens/lewens-repo/lewens-ios/)** – The core iOS source code (SwiftUI, MVVM).
- **[docs/](file:///Users/ake11a/Swift/Lewens/lewens-repo/docs/)** – Project documentation and technical guides.
- **[scripts/](file:///Users/ake11a/Swift/Lewens/lewens-repo/scripts/)** – Development scripts for auth and setup.
- **[config/](file:///Users/ake11a/Swift/Lewens/lewens-repo/config/)** – Certificates and provider jars.
- **[design/](file:///Users/ake11a/Swift/Lewens/lewens-repo/design/)** – Logos and design assets.

## 📖 Key Documentation

- **[Performance & Optimization](file:///Users/ake11a/Swift/Lewens/lewens-repo/docs/PERFORMANCE.md)** – Startup and UI responsiveness.

## 🚀 Getting Started

### iOS
1. Open `lewens.xcodeproj` in **Xcode 14+**.
2. Select the `lewens` target.
3. Build and run on **iOS 16+**.

### Android
*The Android project scaffolding is prepared and will be added under the `Android/` directory.*
