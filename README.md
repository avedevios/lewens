Lewens is a cross‑platform mobile application targeting iOS and Android.

This repository currently contains the iOS project. The Android project scaffolding is prepared and will be added under the Android directory.

Directory layout
- iOS – placeholder directory for iOS-specific assets/docs (current iOS code is in lewens and lewens.xcodeproj)
- Android – placeholder directory for the Android project (to be added)
- design – logos and design references

Getting started
- iOS
  - Open lewens.xcodeproj in Xcode 14+.
  - Build and run the lewens app target on iOS 16+.
- Android
  - The Android project will be provided in the Android folder.

Design assets
- The repository includes design resources in the design folder.
